# Pixel

stash 프로젝트의 프론트엔드 애플리케이션

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 7
- **Language**: TypeScript 5.9
- **Node.js**: 25.x

## Getting Started

### Prerequisites

- Node.js 25.x

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Project Structure

```
src/
├── assets/          # 정적 파일
├── components/      # 컴포넌트
├── App.tsx          # 메인 앱
└── main.tsx         # 진입점
```
